使用简介：
1. 在使用该日期控件的文件中加入JS库(仅这一个文件即可,其他文件会自动引入,请勿删除或改名), 
代码如下 <script language="javascript" type="text/javascript" src="My97DatePicker/WdatePicker.js"></script>
注:src="My97DatePicker/WdatePicker.js" 请根据你的实际情况改变路径

2. 加上主调函数 WdatePicker
<input type="text" id="d243" onfocus="WdatePicker({skin:'whyGreen',dateFmt:'yyyy年MM月dd日 H:mm:ss'})" class="Wdate"/>
